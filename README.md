## 🎨 The Aesthetic Philosophy
Frequency 2004 is a nostalgia machine meticulously designed to recreate the specific visual texture of browsing the web in the early 2000s. The design blends elements from **MySpace (2003-2008)**, **Orkut (2004-2010)**, and the classic **Windows XP / Internet Explorer 6** system interface.

*   **VCR OSD Typography**: Features a system-wide font configuration utilizing `VCR OSD Mono` (VCR EAS) to mimic retro VHS and CRT monitors.
*   **XP Bliss Windows**: Metallic borders, beveled outset/inset panels, and blue-to-gradient title bars.
*   **CRT Scanlines**: Subtle styling overlay elements that make the application feel like a physical 2004 monitor screen.
*   **Sparkle Cursor Trail**: Interactive mouse coordinate tracker emitting trailing custom colored stars.

---

## 🚀 Key Features

*   **⚡ WebSockets Shoutbox**: Real-time global discussion board broadcasting message feeds instantly to online clients with transparent HTTP AJAX polling backup.
*   **📂 Multi-Source Custom Avatars**: Choose from 16 custom-made 8-bit retro pixel icons or upload a custom image file natively using backend storage engine hooks.
*   **🎵 iPod click-wheel Music Player**: Authentic interactive iPod interface equipped with Bollywood classics. Features ambient profile-specific background audio autoplay.
*   **👥 Community Forums**: Create and customize discussion communities. Owners can edit community descriptions, categories, and logos dynamically.
*   **💬 Private Messaging (DMs)**: Chat in private windows featuring retro style, chat nuke reactions, and live polling updates.
*   **💘 Orkut-style Rating Grids**: Rate friends' Trustworthy, Cool, and Sexy status. Mutual crushes unlock secret pink heart profiles.
*   **📝 Scrapbooks & Testimonials**: Sign scrap walls and submit testimonials that owners can review, approve, or hide.

## ⚙️ Local Setup

### 1. Clone the repository
```bash
git clone https://github.com/mishraansh07/Frequency2004.git
cd Frequency2004
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the development server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your web browser. The server will automatically spin up the database schemas and seed 13 interactive dummy profiles.
